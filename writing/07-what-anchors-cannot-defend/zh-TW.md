# KL 錨守不住的東西

*本系列談的是把世界模型塞進 512 KB 的微型無人機。第 6 篇把五個技能
串成一位飛行員；這一篇談如何在改進技能的同時不把技能改死——微調
安全法則的六條量測條款、一頂拿下的王冠、一頂誠實懸空的王座。
所有數字都能從
[microdrone-world-model](https://github.com/csinghans/microdrone-world-model)
重跑出來。*

---

> **白話版。**跟新員工說「做法不要偏離老師傅太多」，然後讓他整個月只操練一項新任務——一千步「每步都沒偏太多」之後，他做起事來已經是另一個人。這就是帶 KL 錨的微調：鏈子綁的是每一步、不是你最後走到哪。本篇是「升級一個技能而不磨掉其他技能」的量測版安全法——外加誠實的一句：有些東西，任何錨都守不住。


## 問題：修復而不抹除

行為克隆給了我們一位裝在 400 KB 網路裡的五技能飛行員。其中兩個
技能偏弱（gap 0.70、moving-gap 0.43，bar 各是 0.75 與 0.70），
顯然該做的事是 PPO 微調。顯然該做的事以很有教育意義的方式失敗
了：從 25k 到 450k 步的**每一個劑量**，弱技能都痊癒、而那個 RL
學不會的技能（四十個決策的蛇行長鏈）都死——25k 步內死透，比修復
完成得更快。先蝕掉的東西，早停救不了：朝 RL 自己吸引子的重新
最佳化是立刻開始的。

標準解法是 KL 錨：在每個訓練狀態上懲罰對凍結 prior 的偏離。
常數係數 1.0 真的解耦了——鏈保住 0.933、gap 修過 bar——但
moving-gap 卡在 0.43：它需要的修復距離就是比 1.0 球的半徑遠。
**常數錨是交換**：選一個係數，等於選一個要輸掉的技能。

## 第一課：排程買得到常數換不到的

撥盤戰役讓兩臂考同一份三 bar 考卷。常數 kl=0.3：鏈 0.833、gap
0.933、moving-gap 0.567——差一個 bar，而且付的鏈稅跟贏家一樣多。
**線性排程 1.0→0.1：鏈 0.833、gap 1.000、moving-gap 0.800——
三 bar 全過。** 早段的緊，護送脆弱技能撐過重新最佳化壓力最兇的
時期（劑量曲線早就量過：腐蝕立刻開始）；晚段的鬆，補完緊球搆
不到的修復。這顆排程工件接著拿下了 corridor-slalom-v2 的王座
——懸空十一考、五個 RL 旋鈕家族、一堵蒸餾牆——合併 84/120 =
0.700、精確等號、guard 全綠。零邊距跟王冠一樣寫得明明白白。

## 第二課：錨只保護 rollout 質量所到之處

第一次晉升就死在一個訓練時沒人量過的 guard 上：雙倍速的
fast-solo 格。prior 飛它 crash 3%；錨定微調後 **31.7%**——十倍
腐蝕、直接穿過 kl=1.0 的錨。機制量出來之後簡單得難堪：KL 項算
在 rollout 態上，而那個切片只占 rollout 質量的 ~4%。與訪問量
成正比的保護＝沒有保護。一個旋鈕之後——`edge_bias`，把一半訓練
集數抽向包絡線的快端——切片質量拉了 ~6 倍、腐蝕消失：31.7% →
5.0%，還帶著全卷最肥的淨空。**第五條款：錨壓是質量加權的；
薄切片腐蝕起來近乎裸奔。**

## 第三課：錨防的是漂移、不是獎勵

然後我們把王冠配方指向 dodgeball 王座——目標行為是**守住站位
框、同時閃球**。鍋克隆在每個球速都守站。排程錨微調把守站抹到
**四速全 0.000**——而站位獎勵全程都在付錢、錨也全程「守著」那個
守站的 prior。

殺它的不是漂移。八個食譜世界裡五個付前進獎勵；這個梯度多數
主動反對留在框裡，而錨在尾段的鬆手恰好在整併期批准了這張票。
**第六條款：錨防得住漂移——獎勵不置可否的行為——防不住獎勵
反對的行為。** 蛇行鏈能在同一配方下活下來，是因為那份食譜的
獎勵沒有任何東西反對繞柱。

逐組係數（ball 態的錨釘在 1.0、transit 態照排程）按比例防守：
球場成功率隨 ball 係數單調——全域排程下 0.000、0.5 地板 0.2-0.4、
釘死 0.33-0.55。但釘死臂在每一格都嚴格支配地板臂，這就把牆的
名字叫了出來：ball 態上的自由買到的是侵蝕、永遠不是修復，因為
**要守的行為與要進步的行為住在同一批狀態上**。我們接著把 ball
態重錨到腳本老師本人（影子神諭逐 rollout 列標籤；對決定性老師
的 cross-entropy 恰好就是那個 KL）——預註冊的處決條款以字面
開火：凍結的 headline 與釘死臂精確平手、21/60 對 21/60。邊註
有動（crash 五倍降、最快格跨過自己的 bar、一個 guard 首綠）——
照記、不翻案。dodge 王座續懸，從五個方向被定完價。

## 法則全文，六條款

對多技能 prior 做微調：(1) 修食譜內的技能；(2) 蝕食譜外的技能；
(3) 連食譜內都抹掉 RL 學不會的技能——腐蝕跑贏修復；(4) 強而不
完美的 prior 可能落進比從零更糟的盆地；(5) 錨的保護是質量加權
的——薄切片近乎裸奔；(6) 錨防漂移、不防獎勵——當守與修住在同
一批態上，逐組釘死用修復換防守。

每一條款都是一場量測戰役、不是一個信念。其中三條是失敗的
奪冠嘗試買回來的——失敗的嘗試本來就是幹這個用的。

## 自己重跑

```bash
# 撥盤：常數 vs 排程（王冠臂）
python -m scripts.distill --finetune <BC2.zip> --steps 450000 \
  --world slalom3_fixed,gap,moving_gap,classic,solo \
  --anchor 1.0 --anchor-end 0.1 --ft-edge-bias --out sched.zip

# 逐組釘死／DAgger 錨（dodge 各臂）
...  --anchor-ball-end 1.0      # 釘死 ball 組
...  --anchor-dagger-ball 1.0   # ball 列的影子老師 CE

# 考卷
python -m eval.eval_policy_cells --zip sched.zip \
  --cells experiments/conservative_ft/index_cells.json \
  --skill corridor-slalom-v2
```

`experiments/anchor_dial/`、`experiments/slalom_v2_promotion/`、
