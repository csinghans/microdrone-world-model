# 那一秒的退縮：解剖一個瞬間

*本系列第 11 篇，主題是一顆塞得進 512 KB 的 latent 世界模型。第 9 篇
跟著一道部署門從 72 走到 79；這一篇只跟著**一秒鐘**——無人機逼近
移動缺口、卻在最後關頭轉頭離開一條明明穿得過的縫的那一秒——穿過
一場六嫌疑人的謀殺調查、一次失敗的師徒制、和一筆稅的命名。所有
數字都能在
[microdrone-world-model](https://github.com/csinghans/microdrone-world-model)
重跑。*

---

> **白話版。**匯入高速公路時，緊張的駕駛有時會在最該踩油門的瞬間
> 踩了剎車——原本進得去的車縫就這樣關上。我們的無人機在移動缺口
> 前做一模一樣的事。這是一個「當場抓到退縮、證明縫本來穿得過、
> 然後發現『不退縮』這門課為什麼教不會」的故事：教練靠著學生的
> 後照鏡解析不了的東西判斷時機。你可以處決其他所有嫌疑人、複製
> 老師的手、直接用成功發獎金——最後仍剩一塊頑固的殘餘，而它有
> 名字：感知稅。

## 第一幕——調查

我們把整合賽道從三關拉到六關、只問一個數字：**0.500**——低於鏈
算術預測的 0.55–0.65，而解剖把超額死亡釘在一個地方：交接失敗率
隨深度上升、第 1 個接點 3.2%、第 5 個 20.6%。有東西在隨深度複利。

六個嫌疑人依序被預註冊的探針或介入處決：**入場姿態**（每個邊界
主動把飛機開回中線——姿態壓到規格、死亡一分未動）；**會合位移**
（我自己最愛的理論——composite 逐關懶推進、儀器一槍斃了它）；
**意外旁路**（稽核為零）；**入場條件**（全深度平坦且不預測）；
**倖存者組成**（上游混合幾乎不變）；**控制器的時間累積器**（每個
邊界歸零——梯度不動）。六次處決、無一定罪。

站得住的是這個：逐 tick 軌跡顯示、死掉的飛行員在逼近中段追蹤移動
縫**追得比活下來的還準**、然後在終端一秒轉往錯的方向——**最後
十二個決策裡九成往錯處動**。一個特權 oracle 定了因果：在最後一
公尺強制 commit、死亡就消失——深縫失敗 **27% → 4.2%**、整場六關
綜合 **0.500 → 0.610**。一個關型的一秒鐘值十一分。退縮是**失誤**、
不是投降：縫從頭到尾都穿得過。

## 第二幕——師徒制失敗

oracle 作弊——它直接讀柱子座標。能出貨的飛行員只有相機。於是走
看家配方：讓作弊的老師飛四百條賽道、只錄學生合法看得到的畫面與
實際動作、再複製這套行為。

資料鍋自己的品質地板拒收、兩次。擬合儀表講得很精確：巡航段複製到
0.948、**終端那一秒 0.856**——而混淆結構是把剃刀：方向搞反的錯誤
**一百零四個裡只有一個**。學生完全知道縫在**哪**；學不會的是
**何時**——老師的規則開在移動縫旁 ±2 公分的邊界上、示範因此高頻
閃爍（轉、直、轉）、任何視覺估計都解析不了。我們試著給老師加
遲滯讓示範平滑、結果反噬得很有教育性：更平滑的 oracle 自己飛得
**更好**（自身任務 +8）、但示範變成依賴歷史——同一張畫面依老師
隱藏狀態對應不同動作——反而更難複製。**病在教法、不在學生。**

照樣飛（擬合地板誠實重註冊、所有飛行判準一字不動），複製人拿到
oracle 收益的一半：縫失敗 17.2% → 11.1%、對上凍結的 ≤8%。真實、
但不夠。

## 第三幕——稅被命名

強化學習不需要一致的示範——只需要結果。穿過去活著有獎、退縮
沒有；策略可以把自己的 commit 邊界放在**它的感知解析得了**的
地方。我們用複製人暖啟動（它知道「哪」）、以加冕配方微調——
KL 錨從 1.0 排程放鬆到 0.1、450k 步、先緊後鬆的牽繩。

一百二十條新鮮賽道彙池：本線史上最佳的畢業綜合分（0.808）、
沒有傷到任何隊友、主判準落在 **8.9% 對凍結的 8%——差十分之四趟、
駁回**。bar 沒有動。但駁回觸發的 review 是預名的、其判詞正是本文
的重點：兩個獨立的、部署合法的方法收斂了——BC 11.1%、RL 8.9%
——對照特權天花板的 ~4%。殘餘不是無能、也不是運氣。**它是感知
稅：oracle 從模擬器直讀的東西、與一顆 64×64 相機在最後一公尺對
移動縫解析得出的東西之間的差。**凍結的目標假設學生能拿到 oracle
收益的四分之三；現實開價 47–63%。

## 一秒鐘教的事

- **介入勝過觀察。**相關性指錯了三次機制；兩次強制介入（回中、
  強制 commit）解決了觀察解決不了的問題。
- **一門課教不會、可以不是任何人的錯。**「現在就衝」這句話無法
  用學生的感官表達；再多示範也修不了、而純獎勵學習恰好只能拿回
  感知允許的份額。
- **判準服務決策；證據活得比判準久。**RL 手臂差 0.4 趟沒過、
  不出貨——但它的量測價值入了庫、未來攻下一道門的疊層會用
  **它自己的考卷**來審。同時「守衛錨要用彙池真實率、永不用單場
  考卷好籤」這條教訓兩週內咬了我們三次才成為常設規則。帳本就是
  老師。

## 重跑這些數字

```bash
python -m eval.eval_integration --suite 100 --contender hybrid --k 6 --seed0 140000
python -m eval.eval_gap_phase --analyze                     # 終端錯向轉彎
python -m eval.eval_integration --suite 100 --contender hybrid --k 6 \
  --seed0 140000 --thread-commit                            # oracle：退縮是失誤
python -m scripts.build_mgap_commit --selftest              # 師徒制的資料鍋
```

門檻、分支表與每一句判詞、都在數字出現之前 commit：
`experiments/integration_k6_v1/`（調查）、
`experiments/terminal_commit_v1/`（師徒制）、
`experiments/mgap_rl_v1/`（稅）。
