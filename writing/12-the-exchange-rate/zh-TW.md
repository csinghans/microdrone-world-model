# 證據的匯率：你量到的數字，有多少是真的？

*本系列第 12 篇，主題是一顆塞得進 512 KB 的 latent 世界模型。第 11
篇解剖了一秒鐘、給一筆稅命了名；這一篇講那個故事裡兩個數字的
**後續**——一個被駁回的 8.9% 變成了六分門檻分、一個嚇人的 3.4×
相關性在介入之下縮成 1.7×。所有數字都能在
[microdrone-world-model](https://github.com/csinghans/microdrone-world-model)
重跑。*

---

> **白話版。**健身房會員比不運動的人健康——隨便一份問卷差距都很
> 大。但「辦卡」**造成**了那個差距嗎？唯一誠實的辦法是做實驗：找
> 一對一對條件相近的人、每對送一個去健身房、然後比較。真的做了，
> 差距通常會縮小——辦卡有用、但大概只有問卷說的一半，因為本來就
> 健康的人才是去辦卡的人。相關性自帶一個匯率，而你不介入就不知道
> 匯率是多少。這篇文章在同一架無人機、同一週裡量了兩次匯率：一次
> 是一個「落榜」的訓練成果在對的考試下值六分；一次是一個 3.4× 的
> 死亡倍率其實只有 1.7×。

## 第一部——落榜的數字兌了現

第 11 篇的結尾，是一條被十分之四趟駁回的強化學習手臂：移動缺口縫
8.9%、對凍結的 8%。bar 沒有動、手臂沒有出貨——但 review 判詞留了
一句很小心的話：*採用交給疊層自己的註冊、用疊層的考試判、不用這
條手臂沒達標的野心判。*

那場註冊跑了。新凍結的 bars（標準 100 賽道考卷 ≥80；每個守衛錨定
彙池**真實率**；換上去的槽必須打敗前任的 0.887 真實條件率、否則
整件事只買到紙面）。把 RL 手臂放進移動缺口槽的陣容考出 **85/100**
——守衛全綠、換的槽 0.9455。部署門從 79 重錨到 85；回頭重量六關
儀器、0.500 → 0.570、深處移動缺口失敗**減半**——跟「學生拿得到
oracle 收益的一半」的算術一分不差。

兩條帳本註記讓這件事誠實。85 是張好籤（約 +1σ）、真實實力
~0.80–0.81——手臂自己的 120 賽道彙池才是真相、考卷是儀式。而且
當初的駁回**沒有錯**：那道 bar 服務了它自己的決策（不在那場註冊
的主張下出貨）、證據活得比它久。一個在某個審判下不及格的數字、
在另一個審判下可以值六分——前提是第二個審判在讀數之前就註冊好。

## 第二部——縮水的相關性

六關的讀數也點了下一個前沿的名：移動缺口治好一半之後，**slalom**
成了最大的失敗型（16.2%）。免費翻一遍紀錄檔、翻出一件搶眼的事：
**誰交棒給你、預測你會不會死。**接在關門後面進 slalom 的、死
9.7%；接在開門後面的、死 **33.3%**——3.4× 的倍率、新鮮考卷上乾
淨複製。

然後每個顯眼的嫌疑人都死在探針下：

- **入場狀態？**橫向速度：完全打平（具名嫌疑、0.88× 開釋）。橫向
  偏移：簽名是真的（z = 5）——但交棒交得**最歪最快**的（slalom
  接 slalom）死得最少，而且把偏移配對之後、開門的超額死亡在窄帶
  內仍然是 3.2×。是標記、不是病因——這個系列第四次抓到「位姿」
  穿這件戲服。
- **死在哪裡？**先踩了一個儀器陷阱：composite 是飛完全程、事後用
  最小淨空判撞——最後一筆記錄不是死亡點（落後 38–173 tick）。
  天真地讀、每個死亡都發生在關卡邊界——而那裡的世界是**空的**。
  正確映射之後：死亡全部落在圍欄上、三根欄平均散佈、每種前手都
  一樣。就是普通的「沒穿過縫」織行失敗——只是開門之後**多**。
  頻率效應、不是形狀效應。而且跟移動缺口的「終端一秒」不同：
  死掉的 slalom 飛行員**全程**偏離 oracle 路線——這個病沒有單一
  的致命瞬間。

所以效應是真的、可複製的、而且不由任何記錄得到的座標承載。剩下
一個介入能終審它到底是不是因果：**同一條賽道飛兩次、只改前一關
的名字。**composite 每一關的幾何恰好消耗一次亂數抽籤，所以同
seed 換掉第 0 關的名字、第 1 關的 slalom **逐位元不變**——同樣的
欄、同樣的縫、位置與組成全部凍結（selftest 證明配對斷言有牙齒：
換個 seed 它就炸）。

301 對賽道之後：開門之後 14.6%、關門之後 8.6%。**比 1.69、
McNemar z = 2.25、跨塊複製（1.71／1.67）。**交接稅是**真的**——
其他一切逐位控制之下、光是「誰交棒」就造成 +6 個百分點的死亡。
而它只有觀察表宣稱的**一半**。另一半是混淆（深度混合和它的朋友
們）穿著前手的戲服。

載體至今未被指認——它住在沒有任何探針記錄的地方，首嫌是幀級
感知：開門交棒之後的那幾秒、相機實際**看到**了什麼。這是一個具
名、未定價的 open；藥（如果有）等它自己的註冊。

## 兩個數字教的事

- **相關性自帶匯率、而且必須實測。**我們的條件表說 3.4×；配對
  介入付 1.69×。任何靠在既有紀錄上做條件化蓋出來的表——包括我
  們自己的——在介入定價之前都該打著這個折扣讀。
- **判準服務決策；證據活得比判準久。**同一個 8.9%、在一場註冊下
  是駁回、在另一場下是 +6 門分。兩個審判互不矛盾——它們問的是
  不同的問題、而且都在讀數之前凍結。
- **儀器會往你最愛的方向說謊。**邊界死亡的幻覺（事後判撞）給了
  我們一個漂亮、錯誤的故事一整個下午。解法是機械性的：把撞擊
  時刻映回軌跡、永遠不要相信最後一行。

## 重跑這些數字

```bash
python -m eval.eval_integration --suite 100 --contender hybrid --seed0 110000   # 85/100
python -m eval.eval_integration --suite 100 --contender hybrid --k 6 --seed0 140000  # 0.570
python -m eval.eval_pose_walk --probe --seed0 157000 \
  --out /tmp/pose.json                                        # 入場狀態＋前手
python -m eval.eval_seam_fidelity --capture --collect-rows --course-k 6 \
  --seed0 140000 --out-json /tmp/p2.json --out-npz /tmp/p2.npz  # 逐 tick 解剖
python -m eval.eval_pred_ab --pairs 160                       # 配對 A/B
```

`experiments/slalom_depth_v1/`（匯率）。
